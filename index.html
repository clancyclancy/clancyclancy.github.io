Graduate Research

Femur fractures are serious injuries with long recoveries and frequent complications. My research tackles a key surgical challenge: aligning broken femurs accurately. I contributed to the development of Robossis, a robotic system built to assist surgeons in femur repair. Unlike typical robots, Robossis delivers high force and precision, crucial for realigning bone fragments.

Using genetic algorithms, I optimized its mechanical design. Testing showed Robossis achieved just 0.31 mm average error and surpassed clinical standards for motion and force. This work, published in the ASME Journal of Medical Robotics, advances safer, more effective robotic surgery.

Personal Project

Computer Vision Gameplay Automation

This project leverages Python, real-time computer vision, A* pathfinding, and a finite state machine to create a high-performance automation framework capable of interpreting and interacting with dynamic screen environments at up to 30 FPS.

Built specifically for Old School RuneScape (OSRS), the system automates a complete fishing–cooking–banking loop in Lumbridge by detecting game elements using YOLO object detection, navigating via template matching and map grid logic, and executing contextual actions based on game state transitions — all without injecting code or reading memory, making it adaptable and non-intrusive.
